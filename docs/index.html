Romulan War

The playable game is built by GitHub Actions and published on the gh-pages branch. This docs/ folder only exists so a mistaken “publish from /docs” setting does not break the build.

Enable the real site

  1. Open Settings → Pages for this repository.
  2. Under Build and deployment, set Source to Deploy from a branch.
  3. Branch: gh-pages · Folder: / (root) · Save.
  4. Wait a minute, then open: https://<user>.github.io/<repo>/

Ensure the workflow Deploy to GitHub Pages has run successfully on main so the gh-pages branch exists.
